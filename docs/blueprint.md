# **App Name**: Collectii

## Core Features:

- Task Management: Create, assign, and track tasks with deadlines, priorities, and attachments.
- Team Coordination: Organize teams and departments with roles and permissions to streamline collaboration and communication.
- Reporting & Dashboards: Generate visual reports and dashboards with key performance indicators (KPIs) to monitor team and project progress.
- AI Task Prioritization Tool: Uses an AI tool to learn user habits and intelligently suggests task priorities based on deadlines, dependencies, and team member availability.
- Google & Notion Integration: Seamless integration with Google Workspace and Notion to centralize workflows and consolidate information.
- Announcements: Publish announcements to specific groups or departments, and pin important announcements to the top of the feed.
- User Authentication: Secure sign-up, sign-in, and password recovery functionality, with support for 'Continue with Google' authentication.

## Style Guidelines:

- Primary color: Black (#000000) for background to convey a modern feel.
- Secondary color: White (#FFFFFF) for text to ensure readability and contrast against the black background.
- Accent color: Minimal blue (#1E40FF) for interactive elements like links and call-to-action buttons.
- Headings: 'Inter' (sans-serif) in bold for a clean, modern, approachable style.
- Body: 'PT Sans' (sans-serif) in light/regular weight for high legibility.
- Use minimal line-style icons, with blue highlights to match the accent color.
- Top navigation bar and footer to stay consistent across all pages. Dashboard uses a sidebar for navigation.
